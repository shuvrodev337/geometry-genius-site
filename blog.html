What is DOM and what is the purpose of DOM?

DOM or Document Object Model is a programming interface for HTML document created by the browser. When a web page is loaded, the HTML document converts to a DOM tree. The purpose of DOM is to give interactivity to web pages.

How will you select HTML elements using DOM? Name the DOM methods?

Useful DOM methods to you select HTML elements are:

  • getElementsByTagName()
  • getElementByID()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

    • What is event bubble?

    Event bubble is a propagation where the event triggers on the innermost target element. Then it successively goes up the DOM tree. On it's way up, it triggers events that are added on the ancestors of that target element.

    What is a callback function and why will you use it?

    A callback function is a function which is passed as an arguement of another function. We use callback function because we can execute that callback function in response to an event.
